# 🗺 Framework Roadmap

The strategic development plan for the `codex-bot` framework.

## ✅ Phase 1: Core Foundation (v0.1.x)
- [x] Basic feature-based architecture.
- [x] Feature discovery service.
- [x] Initial project templates.

## 🚀 Phase 2: Professional Scaffolding (v0.2.0) - CURRENT
- [x] **Interactive CLI Wizard**: Guided project setup.
- [x] **Monorepo Support**: Isolated package structure (`src/<name>/`).
- [x] **Advanced Logging**: Loguru with data masking and interceptors.
- [x] **Universal Runner**: Centralized launch logic in the library core.
- [x] **Manage.py**: Unified project management interface.

## 🏗 Phase 3: Developer Experience & Ecosystem (v0.3.0) - PLANNED
- [ ] **Integrated Debug System**: Built-in ID inspector and system status commands. (Task #001)
- [ ] **In-Bot Guide System**: Interactive navigator for documentation and tutorials. (Task #003)
- [ ] **Framework Education Mode**: Contextual tips based on project settings.
- [ ] **Docker Infrastructure**: Ready-to-use Dockerfile and docker-compose templates. (Task #002)
- [ ] **Strong Typing**: Migration to Enums for states and callback actions.
- [ ] **Advanced Error Recovery**: Enhanced global error handlers with retry logic.
